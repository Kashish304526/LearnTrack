# Smart Study Planner & Task Tracker

A full-stack web application to help students plan daily study tasks,
track streaks, and visualize progress.

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: FastAPI + SQLAlchemy
- Authentication: JWT
- Database: PostgreSQL / SQLite
- Testing: Pytest

## Features
- User registration & login
- Daily task creation
- Study streak tracking
- Progress dashboard

## How to Run
Frontend:
cd frontend
npm install
npm run dev

Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
